Half Wave Controlled Rectifier

INSTRUCTION

  1. Connections are Explained below:
    L1-L2, L4-L5, L6-L7, L7-L8, L9-L10, L10-L11, L11-L12.
  2. L3 is connected to external firing circuit not mentioned here.
  3. Set the peak value of input supply Vm.
  4. Set the value of firing angle α .
  5. Set the values of load resistance RL and load inductance LL.
  6. Set the frequency of input supply fo.
  7. Set the y-axis range Vmax [Vmax = at least 2Vm]
  8. Set the x-axis range Tmax.
  9. Click on Draw Waveform.
  10. Calculate Output Average Voltage \[V_{o,avg} = {V_m \over 2\pi}(cos \alpha-cos \beta)\]
  11. Calculate Output RMS Voltage \[V_{o,rms} = {V_m \over 2\sqrt\pi}[(\beta - \alpha) - {1\over2}(sin 2\beta-sin 2\alpha)]^{1\over2}\]
